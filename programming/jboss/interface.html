3.  Die Benutzeroberfläche

Nach Installation der database-Anwendung und dem Start des JBoss sollte die Web-Adresse http://localhost:8080/database/ existieren.

Hochladen

Da anfangs noch keine Datensätze auf dem Server vorhanden sind, bietet die Anwendung das Hochladen einer ASCII-Datei, die mehrere Datensätze enthält an. Dem Projekt ist unter resources/Datas.txt eine solche Datei beigefügt.



Bild zur Demonstration des Hochladens


Eingabe und Suche

Natürlich können auch über eine Eingabemaske neue Datensätze hinzugefügt werden. Eine zweite Maske zur Suche ermöglicht das Finde einzelnen Daten.



Bild zur Demonstration der Eingabe


Suchergebnisse

Das Resultat der Suche und auch alle Datensätze werden als Tabelle angezeigt. In ihr kann auch das Löschen eines Datensatzes ausgelöst werden.



Bild zur Demonstration der Suchergebnisse
