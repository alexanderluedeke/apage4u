 tag und page context

6.1 page context

Für jede Request existiert ein page context. Durch ihn können auf der Server-Seite Objecte erzeugt und angesprochen werden.

private String jndiName = "Bach";


private Music aMusic;


pageContext.setAttribute(jndiName,aMusic,PageContext.SESSION_SCOPE);

aMusic = (Music) pageContext.getAttribute(jndiName);

So können unterschiedliche tags mit Hilfe des Names auf ein Object zugreifen. Bei SESSION_SCOPE können sich die tags sogar auf unterschiedlichen jsp-Seiten befinden.
